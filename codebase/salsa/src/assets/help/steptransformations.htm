Step Transformations

A step consists of an input table(s) and a transformation taken on that table(s). The result creates a derived table. This table will appear in the left panel along with the other tables in the Flow. You can use the created derived table as an input table in any step.

You can choose from six different transformations. Each transformation will have its own specific set of inputs. However, each will require you to name the output table.
