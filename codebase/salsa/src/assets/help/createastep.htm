Create a Step

Flows are made up of a series a Steps. A Step is where you execute transformations on tables to produced derived tables.  Each Step can only contain 1 data engineering operation. Operations can be thought of as appending a new column to the existing table structure. When the Data Engineering operation executes successfully, it will be added to the Flow List and, optionally, the DERIVED folder in the LibraryTo add a Step, click on the “Add Step” icon in the left panel.

After you click Add Step, you will see a set of Transformers you can choose from.
