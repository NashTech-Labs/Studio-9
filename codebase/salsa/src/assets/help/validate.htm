Validate

Gives the user the ability to test model performance against either a subset of the training data (e.g. testing out of bag performance via an 80-20 split or testing model performance on a subset of data of interest) or the entire dataset used for training as long as the new dataset has the exact same data structure (same number of columns, with data types, relation, etc.).  This allows the user to compare actual and predicted responses for the executed dataset by mapping each source table and column to the corresponding new table and column.
