Tables

There are various types of Tables in inSilico. Regardless of the type they will all be stored here. These are all the tables of data, and their corresponding Meta-Data, to which the user has access and is permitted to utilize. Tables are used to create Flows, A.I. models or to create Dashboards & Reports.

The different types of tables are:

1.  Source Table – Tables that you imported into inSilico

2.  Derived Table – Tables that were generated as a result of a Transformation. This includes Tables that were created in Compose, Train and Validate
